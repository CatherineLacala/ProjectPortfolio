♠️ 3-Card Poker Client-Server Application

Java | JavaFX | Networking

Multiplayer 3-Card Poker game built with a client–server architecture supporting real-time gameplay between multiple players.

Key Features:

- Concurrent socket-based networking for multiplayer support

- Event-driven user interface built with JavaFX

- Object-oriented game architecture and modular design

- Real-time game state synchronization between clients and server

Concepts Demonstrated:

- Network programming with sockets

- Concurrent programming

- Object-oriented design patterns

- Event-driven UI development
